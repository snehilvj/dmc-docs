### New TableofContents Component

Use the `TableOfContents` component to display a table of contents similar to the sidebar in these docs. The
component tracks scroll position and highlights current heading in the list.

Check out all the new features in the [Table Of Contents docs](/components/table-of-contents).

*This component was added by @deadkex in [PR #513](https://github.com/snehilvj/dash-mantine-components/pull/513)*

### Custom DMC components

`MantineHooks` and `MantineCore` are now exported from dash-mantine-components, giving component authors access to the same hooks and utilities DMC uses internally.

This allows you to make custom components, such as a `Select` built on lower-level building blocks like [Combobox](https://mantine.dev/core/combobox/), while maintaining consistent theming in Dash.

For more details and examples, see custom DMC components based on  [Mantineâ€™s custom Combobox examples](https://mantine.dev/combobox/?e=BasicSelect) in this [GitHub repo](https://github.com/AnnMarieW/dmc_custom_components).

*This feature was added by @BSd3v in [PR# 653](https://github.com/snehilvj/dash-mantine-components/pull/653)*  

### Improved typing

The Python types for DMC components have been improved to better reflect supported props and values. This provides more
accurate type hints, better autocomplete, and clearer static checking in editors like VS Code and PyCharm.
